Welcome to Grade-Boundary Explorer

This site offers a free, open-access way to analyse and compare A-Level grade boundaries for Edexcel, CIE, Oxford AQA, and OCR. Access tables, download CSVs and visualise trends with interactive charts.

📢 If you found this website helpful, please consider sharing it — your support would mean a lot!

Available datasets

Edexcel

Over 800 units covering Maths, Sciences, Economics and more (2015-2025).

About This Project

This little site was built by two A‑Level students as a fun project to visualise and compare A‑Level grade boundaries. We’re both studying A‑Levels ourselves, so we know how useful it can be to see trends across exam boards. Enjoy!
